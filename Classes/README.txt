Introduction
============

The STD Guide was the first prototype done to take the information from the CDC STD Guide and put it in a mobile application. After the first review the project was taken in a different direction and development on this version was halted.

Installation and Building
=========================

The STD Guide source code was built using Xcode 4.3.2 running on Apple Mac OS X Lion 10.7. The application was targeted for use on the iPhone or iPod Touch iOS devices. 

Known Issues
============
Since the development of this app was halted in the very early stages there are a lot of known issues with this app.
